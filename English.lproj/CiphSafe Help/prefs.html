CiphSafe Preferences Help

Several behaviors are customizable in CiphSafe. These are broken down into three areas: Interface, Generating Passwords, and Miscellaneous.

Interface Preferences

Close add window after adding an entry
When adding a number of entries at once, disabling this option will make life drastically easier. The add window will stay open, and focused, after an entry is added. If enabled, adding an entry will cause the add window to close.
Close edit window after modifying an entry
Like the equivalent for the add window above, enabling this option causes the view/change window to close after an entry has been modified.
Confirm entry deletion
Asks, "Are you sure?" prior to deleting any entries, when enabled.
Warn about a short passphrase
If checked, will warn when initially selecting a passphrase (or when changing it) when that passphrase is short (less than eight characters).
Create new document on activate
If on, initially starting CiphSafe, as well as clicking its icon in the Dock when it has no open documents, will cause it to create a new document (like TextEdit does). If you find this annoying, turn it off.
Include password on copy/drag to other apps
When enabled, a cut, copy, or drag of any entries will include the password if the destination is an app other than CiphSafe. In the disabled state, all fields except for the password are available to the other application. When a cut, copy, or drag has a destination inside CiphSafe, the password is always copied.

Generated Passwords

Password length
Specify the length for passwords generated by CiphSafe (any value between 1 and 255 is acceptable).
Use only alphanumerics
Will use only letters and numbers (a-z, A-Z, and 0-9) when checked, otherwise will use a broader set of characters from the ASCII set.

Miscellaneous

Keep backup copy
If on, when a document is saved, the previous version of the document will be kept with a ~ (tilde) appended to the name, prior to the extension (eg, document.csd will have document~.csd as the backup name).
Clear clipboard on Quit
Clears the clipboard when CiphSafe quits, if CiphSafe was the last to cut/copy something there.
