Using Drag and Drop

Drag and drop is simple with CiphSafe, as it should be. There are only a few bits you may need to know:

  • when dragging to another CiphSafe document, all fields are preserved properly
  • when going to another application, the password is included only if the preference Include password on copy/drag to other app is enabled under the Security tab
  • when going to another application, CiphSafe will offer two formats: plain text and RTF. If the application handles RTF (e.g., TextEdit with an RTF document), the notes section will be properly preserved. Formatting will be lost if the plain text version is accepted by the destination application.
  • entries are always copied when dragged to an application other than CiphSafe; when another CiphSafe document is the target, however, you can press the Command key to move the entries instead of copying
