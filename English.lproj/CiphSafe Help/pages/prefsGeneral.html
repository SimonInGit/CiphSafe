Setting General Preferences in CiphSafe

The General preferences pane controls some miscellaneous settings for CiphSafe.

  • The "Close add window after adding an entry" and "Close edit window after modifying an entry" options allow you to specify whether the two types of entry windows close after an addition or change is made.
  • The "Confirm entry deletion" option controls whether or not CiphSafe will warn every time you attempt to delete an entry.
  • The "Create new document on activate" option, when enabled, will create a new empty document if CiphSafe is launched without a document specified, or the Dock icon is clicked and no documents are currently open.
  • The "Open document when program starts" option (and accompanying file selection) allows the option to have a selected document open automatically when simply launching CiphSafe.
  • The "Keep backup copy" option will, when enabled, create a file (with a tilde, ~, at the end of the filename) of the previously-saved document, in case you wish to keep an older version around when updating.
  • The "Include default categories" option adds Banking, Forum, General, Other Web, and Retail to the list of categories available, and are always there.
