What's New in CiphSafe 1.3

Several by-popular-request items have been added to CiphSafe for 1.3.

Import/Export

The ability to import/export comma-separated value (CSV) files has been added. Now you can get data out of CiphSafe when needed as well as bring in other stuff.

Using Import/Export


New file format

The file format for CiphSafe has been changed. The previous format basically took the data from Cocoa's frameworks, but now uses a text-based format in case Apple ever changes the way Cocoa handles things. This does have consequences if you want to keep older versions of CiphSafe around.

Information on version compatibility


Drag and drop handles moving and copying

Prior versions only did a copy when dragging entries. Now, per the Mac OS standard key combination of Command to move or Option to copy, entries can be moved or copied as needed.


Alternate row color is selectable

The main view of entries has had the alternating color option for some time, but 1.3 now allows the alternate color to be changed in case the standard blue is too boring.

Using SOME OTHER FEATURE


Additional new features

Some other, minor features include:

  • Searching—With Mac OS X 10.3, the rounded search field became available to non-Apple developers; hence CiphSafe now uses one instead of the uglier set of pop-up menu, rounded text field, and reset button. The option of searching all fields is now also available.
  • Help—CiphSafe's help has been moved to the task-based style popular with Apple's apps. Thanks to Andy Matuschak for the documentation and toolkit.
  • Main window—the spacebar, when the main table has focus, now pages down.
  • Links—Point things to SourceForge instead of withay.com.
  • Code—Not something the average user will see, but a number of changes at the source level have been made to improve things.
