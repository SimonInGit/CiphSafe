Generating a Password

When storing account/password information in CiphSafe, needing to remember a password is unnecessary. Hence, for additional security, CiphSafe can generate a random password for you.

To generate a random password:

  1. In the add or view window for an entry, click the Generate button next to the password entry
  2. To change the generated password's length, open CiphSafe's preferences, select the Security tab, and enter a number for the Generated password length
  3. To select between alphanumerics (A-Z, a-z, and 0-9) or those and other useful characters (dollar sign, slash, etc), open CiphSafe's preferences, select the Security tab, and check or uncheck the Use only alphanumerics for generated passwords checkbox as desired
