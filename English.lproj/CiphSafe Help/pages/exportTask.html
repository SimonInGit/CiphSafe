Using Export

Exporting data is simple: File > Export... is used to export all entries, or you can export only selected entries by selecting them and using File > Export Selected Items...

To export to CSV:

  1. Open a CiphSafe document
  2. Select File > Export...
  3. Select Comma-Separated Value (CSV) for Format
  4. Check the Add header line in CSV if you want the first line to include the field header names (Name, Account, etc.)
  5. Give the filename a name and hit Save
